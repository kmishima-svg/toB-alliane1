セットアップ手順

1. Anthropic APIキー: console.anthropic.com → API Keys → Create Key
2. Google Cloud: console.cloud.google.com → 新規プロジェクト作成
3. 「APIとサービス」→ Google Calendar API と Google Sheets API を有効化
4. 「OAuth同意画面」を設定(テストユーザーに自分を追加)
5. 「認証情報」→ OAuth クライアントID作成(種類:ウェブアプリケーション)
6. 承認済みJavaScript生成元に、GitHub PagesのURL(例: https://yourname.github.io)を追加
7. 発行されたクライアントIDを上の欄に入力
